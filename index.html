Introduction to Data Science & Machine Learning

Data Science is an interdisciplinary field that uses scientific methods, processes, algorithms, and systems to extract knowledge and insights from structured and unstructured data.

Machine Learning, a subset of AI, allows systems to learn and improve from experience without being explicitly programmed.

Why Learn Data Science?

  • High demand in various industries
  • Opportunities in AI and automation
  • Data-driven decision-making

Introduction to Python

Python is a powerful programming language widely used in data science.

1. Print Statement

The print() function is used to display output.

print("Hello, Data Science!")

2. Variables

Variables store data. Example:

x = 5
y = "Hello"
print(x)
print(y)

3. Data Types

  • int: Integer (e.g., 5, -3)
  • float: Decimal (e.g., 3.14, -0.5)
  • str: String (e.g., "hello")
  • bool: Boolean (e.g., True, False)

4. If-Else Condition

The if statement checks conditions.

x = 10
if x > 5:
    print("x is greater than 5")
else:
    print("x is 5 or less")
